Catalogue de pièces illustrées (IPC) – interface utilisateur

L'interface utilisateur IPC se compose de deux espaces de travail : 3D/2D et table DPL. Chaque espace de travail possède ses propres éléments de commande décrits ci-dessous.

L'IPC peut se trouver dans les représentations suivantes :

    2D et 3D permet de visualiser des images 2D et 3D de l'IPC. Dans ce mode, l'IPC comporte un bouton (graphiques 2D ) pour basculer entre deux fenêtres.

    3D seulement permet de ne visualiser que les images 3D de l'IPC. Dans ce mode, le bouton graphique 2D n'est pas disponible.

    2D seulement permet de ne visualiser que les images 2D de l'IPC. Dans ce mode, l'IPC ne contient que les feuilles qui comportent des images 2D. Les autres feuilles et le bouton Graphique 2D ne sont pas disponibles.

Fenêtre 3D/2D

La fenêtre 3D affiche une image 3D de la feuille active. Vous pouvez naviguer dans la fenêtre 3D à l'aide de la souris ou du clavier. Un positionneur (trièdre d'axe) dans le coin supérieur droit de la fenêtre 3D permet de déterminer l'orientation du modèle 3D par rapport aux axes.

La fenêtre 2D affiche des images vectorielles et tramées associées à la feuille active.

Interface utilisateur Bouton Objectif
Feuille   Permet de sélectionner la feuille détaillée dans la liste des feuilles disponibles.
Réinitialiser Rétablit la vue initiale sur la feuille active.
Graphiques 2D Permet de basculer entre les fenêtres 2D et 3D. Le bouton n'est disponible qu'en mode d'affichage 2D et 3D.
Impression Graphique Permet d'imprimer l'image 3D/2D active telle qu'elle s'affiche dans la fenêtre 3D/2D.
Ignorer transparence (pour la fenêtre 3D) Si cette option est activée, permet de cliquer à travers les objets transparents dans la fenêtre 3D, car les objets transparents ne réagissent plus au clic de l'utilisateur.
Tout rétablir (pour la fenêtre 3D) Affiche tous les objets masqués et semi-transparents.
Omettre l'animation (pour fenêtre 3D) Si cette option est activée, permet d'éviter une transition fluide (animation) d'une feuille à l'autre et rend la transition entre les feuilles instantanée.
Paramètres Afficher ou masque la liste des paramètres.

Vitesse d'animation de transition. Ajuste une vitesse de lecture pour la transition de l'explosion.

Activer le contrôle Animation de transition. Si sélectionné, active le curseur qui vous permet de vérifier manuellement les transitions entre les feuilles. Pour afficher le curseur, déplacez le pointeur de la souris au-dessus du bas de la fenêtre 3D, ou tapez en bas de la fenêtre si vous avez un écran tactile.

Eclairage de scène. Définit le nombre de sources de lumière et leur emplacement pour éclairer le modèle dans la fenêtre 3D. Les positions relatives des sources de lumière sont affichées à droite de la liste. Toutes les sources de lumière se déplacent avec l’afficheur.

Afficher le cube de navigation. Affiche ou masque le cube de navigation dans la fenêtre 3D. Le cube de navigation comporte trois types de zones de HotSpot : arête, coin et face. Lorsque vous positionnez le pointeur sur l'arête, le coin ou la face du cube de navigation, la zone correspondante est mise en surbrillance. Avec ces zones, vous pouvez basculer entre les vues standard et effectuer des rotations.

Afficher les arêtes de surface. Si sélectionné, affiche les contours de surface dans la fenêtre 3D.

Anticrénelage. Active une technique pour minimiser les artefacts de distorsion dans la fenêtre 3D.

Occlusion ambiante. Active l'ombrage de l'occlusion ambiante et la technique de rendu dans la fenêtre 3D.

Mode de sélection. Spécifie la représentation des objets pour la sélection dans la fenêtre 3D.

Contraster avec une couleur – la sélection est mise en surbrillance.

Radiographier les objets sélectionnés – les objets apparaissent gris et translucides, à l'exception de la sélection.

Coque transparente – la coque externe du modèle entier apparaît translucide, les objets internes non sélectionnés sont masqués.

Entourer les objets survolés. La sélection est entourée.

Mise en surbrillance   Lorsque vous sélectionnez un objet ou que vous pointez vers un objet dans la fenêtre 3D ou une légende dans la fenêtre 2D, il est mis en surbrillance. Le numéro d'objet (ou CSN) s'affiche sous forme de fenêtre popup pour la sélection. Le schéma de couleurs peut être modifié par l'auteur du catalogue. Le schéma de couleurs suivant est appliqué par défaut :

Jaune — pour un objet ou une légende sélectionné(e). La ligne DPL correspondante est également sélectionnée et de surbrillance dans la table DPL.

Vert — pour la légende au-dessus de laquelle se trouve le pointeur. La ligne DPL correspondante est également mise en surbrillance dans la table DPL.

Barre de vue 3D

Boutons de commande des modes d'affichage dans la fenêtre 3D.

Interface utilisateur Bouton Objectif
Définir le centre Permet de spécifier le centre de rotation du modèle 3D.
Afficher le centre de rotation Permet d'afficher ou de masquer le centre de rotation.
Ajuster tout Permet de rendre l'ensemble du modèle 3D entièrement visible dans la fenêtre 3D.
Aligner l'horizon Positionne l'axe horizontal (mais pas longitudinal) de la caméra parallèlement au plan horizontal dans le système de coordonnées.
Afficher les axes de coordonnées Affiche ou masque les axes d'orientation dans la fenêtre 3D.
Mesurer Ouvre l'outil permettant de mesurer une distance. Dans la fenêtre 3D, cliquez simplement une fois sur un point, puis cliquez à nouveau sur le deuxième point. La distance entre les deux points rouges doit ensuite s'afficher. Pour obtenir la distance entre les points projetés sur le plan vertical ou horizontal, sélectionnez le nom du plan de projection dans la liste. Pour modifier l'unité de longueur, sélectionnez Unités. Pour définir la longueur des lignes de référence, faites glisser le curseur H. Sélectionnez Garder visible pour rendre la ligne de dimension visible après avoir fermé l'outil de mesure. Pour définir de nouveaux points, cliquez sur Effacer ou appuyez sur la touche Echap, puis spécifiez des points dans la fenêtre 3D.

WARNING

N'utilisez pas les résultats de mesure directement dans le traitement car l'illustration 3D peut différer du produit réel. Pour obtenir les dimensions nécessaires à la fabrication ou à la maintenance, reportez-vous à la documentation technique appropriée.

Plan de section Affiche le modèle comme s'il avait été coupé par un plan. Utilisez les contrôles pour définir (Gauche, Haut, Avant) et faire pivoter (X et Y) le plan de section. Pour déplacer le plan de section, faites glisser le curseur D. Utilisez Inverser pour mettre en miroir la section. Sélectionnez le Mode Précis pour améliorer la précision des curseurs. Sélectionnez Garder visible pour rendre la section visible après avoir fermé l'outil. Pour déplacer le plan de section vers le centre de la sélection, cliquez sur Réinitialiser.

Plusieurs modes d'affichage permettent d'afficher le modèle 3D à partir d'une position spécifiée :

Interface utilisateur Bouton Objectif
Face Affiche le modèle vu de l'avant.
Précédent Affiche le modèle vu de l'arrière.
Gauche Affiche le modèle vu du côté gauche.
Droite Affiche le modèle vu u côté droit.
Haut Affiche le modèle vu du haut.
Bas Affiche le modèle vu du bas.
Isométrique Affiche le modèle dans une vue isométrique.
Barre de sélection

Permet d'ajuster, d'afficher, de masquer et de rendre transparents les objets sélectionnés dans la fenêtre 3D.

Interface utilisateur Objectif
Afficher les objets sélectionnés dans une fenêtre distincte Si cette option est activée, la sélection s'affiche dans la fenêtre 3D secondaire.
Ignorer la transparence Si cette option est activée, permet de cliquer à travers les objets transparents dans la fenêtre 3D, car les objets transparents ne réagissent plus au clic de l'utilisateur.
Tout restaurer Affiche tous les objets masqués et semi-transparents.
Ajuster Ajuste à la fenêtre 3D le cadre de délimitation des objets sélectionnés.
Afficher Rend visible les objets sélectionnés qui étaient masqués précédemment dans la fenêtre 3D. La case à cocher de la ligne DPL correspondante est sélectionnée.
Masquer Masque les objets sélectionnés dans la fenêtre 3D. La case à cocher de la ligne DPL correspondante est désélectionnée.
Ne rien sélectionner Permet de désélectionner tous les objets.
Transparence Permet de définir la valeur de transparence pour les objets sélectionnés. La commande Transparence est disponible si un ou plusieurs objets sont sélectionnés dans la fenêtre 3D ou si une ou plusieurs lignes DPL sont sélectionnées dans la table DPL.
Fenêtre Table DPL

Affiche la table DPL. Notez que la barre Table DPL peut contenir un ensemble de filtres (par exemple : UCA, modèle) pour filtrer les données DPL selon les critères sélectionnés. L'utilisation d'un filtre plutôt qu'un autre dépend du standard d'informations utilisé pour créer cet IPC.

Interface utilisateur Bouton Objectif
Imprimer le tableau Permet d'imprimer la table DPL.
Imprimer composite Permet d'imprimer la table DPL et toutes les images 2D disponibles dans l'IPC. Si Table complète est sélectionné, la table DPL complète et toutes les illustrations techniques sont imprimées. Si Table complète est désélectionné, seule la feuille DPL active et l'illustration correspondante sont imprimées. Chaque image 2D (illustration) est imprimée sur une page distincte.
Table complète   Affiche la table DPL complète. Le texte des lignes qui ne sont pas associés à la feuille active (lignes inactives) est gris.
Exporter vers CSV Exporte les données de toutes les lignes DPL sélectionnées dans un fichier dont les valeurs sont séparées par des virgules. Si aucune ligne DPL n'est sélectionnée, seules les lignes DPL associés à des éléments 3D visibles sont exportées.
Aide Fournit un système d'aide pour l'interface utilisateur.
Zone de titre   La zone Titre contient des informations sur l'IPC, par exemple : nom de la publication, nom de la feuille, date de publication.
Colonne Cases à cocher   La colonne Cases à cocher est une colonne DPL avec des cases à cocher. Les indications suivantes sont utilisées :

 : l'élément correspondant de la ligne DPL est visible dans la fenêtre 3D sur la feuille active.

 : signifie que l'élément correspondant est masqué dans la fenêtre 3D sur la feuille active.

L'absence de case à cocher signifie que la ligne DPL ne comporte pas d'élément 3D correspondant sur la feuille active.
Mise en surbrillance   Lorsque vous sélectionnez une ligne DPL ou que vous pointer vers une ligne DPL, il est mis en surbrillance. Le numéro d'objet (ou CSN) s'affiche sous forme de fenêtre contextuelle pour la ligne DPL mise en surbrillance. Le schéma de couleurs peut être modifié par l'auteur du catalogue. Le schéma de couleurs suivant est appliqué par défaut :

Aigue-marine — pour la ligne DPL au-dessus de laquelle se trouve le pointeur et qui fait référence à des objets illustrés. Les objets et les légendes sont également mis en surbrillance dans la fenêtre 3D/2D.

Jaune — pour la ligne DPL sélectionnée. Les objets et les légendes correspondants sont également mis en surbrillance dans la fenêtre 3D/2D.

Rose — pour la ligne DPL au-dessus de laquelle se trouve le pointeur et qui n'a pas d'objets illustrés.

Menus contextuels dans les fenêtres 3D/2D et Table DPL

Cliquez sur un objet dans la fenêtre 3D ou sur une ligne DPL dans la table DPL ou sur une légende dans la fenêtre 2D pour accéder aux menus contextuels.

Interface utilisateur Définition
Élément/CSN Affiche le numéro d'élément ou un numéro de séquence de catalogue (CSN) pour l'élément, la ligne DPL ou la légende sélectionné.
Ajuster Ajuste le cadre de délimitation des objets sélectionnés à la fenêtre 3D.
Afficher/Masquer Affiche ou masque l'élément sélectionné dans la fenêtre 3D. La case à cocher de la ligne DPL correspondante est respectivement sélectionnée ou désélectionnée.
Sélectionner/Désélectionner Sélectionne ou désélectionne l'élément sélectionné dans la fenêtre 3D, la ligne DPL sélectionnée dans la table DPL ou la légende sélectionnée dans la fenêtre 2D.
Liste de feuilles Permet de sélectionner une feuille dans la liste des feuilles disponibles illustrant la ligne DPL choisie. La feuille recommandée pour la ligne DPL est affichée en gras. La feuille actuelle est désignée par le symbole de coche.
Afficher tout Affiche tous les objets masqués dans la fenêtre 3D et coche les cases correspondantes dans la table DPL.
Zoom avant Permet d'augmenter l'image.
Zoom arrière Permet de réduire l'image.
Ajuster Ajuste l'image à l'intérieur des limites de la marge.
A propos Affiche les versions d'IPC, de l'afficheur 3D et de l'afficheur 2D.

Fenêtre 2D – principales procédures

Les légendes dans la fenêtre 2D sont des Hotspots, qui permettent l'interaction entre les objets et les lignes DPL correspondantes. Cliquez avec le bouton droit dans la fenêtre 2D pour accéder aux commandes contextuelles disponibles. Cliquez avec le bouton droit sur une légende pour accéder aux commandes contextuelles associés à l'objet (les commandes sont décrites dans la section "Menus contextuels dans la fenêtre 3D/2D et dans la table DPL")

Opération Action Utilisateur
Sélectionnez une légende d'objet Cliquez sur la légende nécessaire.
Panoramique Déplacez la souris tout en maintenant le bouton de gauche enfoncé.
Zoom Tournez la molette de la souris vers l'avant ou l'arrière.
Fenêtre 3D – principales procédures
Opération Action Utilisateur
Sélection d'un objet Cliquez sur l'élément nécessaire.

Zoom +/- Tournez la molette de la souris vers l'avant ou l'arrière. Sur les ordinateurs Apple, utilisez la touche modificateur de contrôle.
Panoramique Déplacez la souris avec la molette de la souris enfoncée. Sur les ordinateurs Apple, utilisez la touche modificateur d'option.
Tourner Déplacez la souris tout en maintenant le bouton de gauche enfoncé.

Ou en maintenant la touche Alt enfoncée, cliquez sur un objet dans la fenêtre 3D.
Ajuster le modèle 3D Dans la barre Vue 3D, cliquez sur Ajuster .
