附图零件目录 (IPC) - 用户界面

IPC 用户界面由两个工作区组成：3D/2D 和 DPL 表。每个工作区都有其自己的控制元素，如下所述。

IPC 可以采用以下表示：

    2D 和 3D 支持同时查看 IPC 的 2D 和 3D 图像。在这种情况下，IPC 具有一个用于切换这两个窗口的按钮（2D 图形 ）。

    仅 3D 仅支持查看 IPC 的 3D 图像。在这种情况下，“2D 图形”按钮不可用。

    仅 2D 仅支持查看 IPC 的 2D 图像。在这种情况下，IPC 仅包含具有 2D 图像的工作表。其他工作表和“2D 图形”按钮不可用。

3D/2D 窗口

3D 窗口显示当前工作表的 3D 图像。可以使用鼠标或键盘在 3D 窗口中导航。使用 3D 窗口右上角的定位器（三向轴），可以确定 3D 模型相对于轴的方向。

2D 窗口显示与当前工作表关联的矢量和光栅图像。

用户界面 按钮 用途
页   用于从可用工作表列表中选择详细工作表。
重置 恢复当前工作表的原始视图。
2D 图形 用于在 2D 和 3D 窗口之间切换。该按钮仅在 2D 和 3D 查看模式下可用。
打印图形 用于打印 3D/2D 窗口中显示的当前 3D/2D 图像。
忽略透明度（对于 3D 窗口） 如果启用，就可以穿过 3D 窗口中的透明项执行单击操作，因为透明项将不对用户的单击做出响应。
全部恢复（对于 3D 窗口） 显示所有隐藏和半透明对象。
跳过动画（对于 3D 窗口） 如果启用，就可以避免从一个工作表到另一个工作表的平滑过渡（动画），并在工作表之间进行即时过渡。
设置 显示或隐藏设置列表。

转换动画速率。调整爆炸转换的回放速度。

启用转换动画控制。如果选中，将会激活滑块，用于手动查看工作表之间的转换。要显示滑块，请将鼠标指针移到 3D 窗口底部，或者在使用触摸屏的情况下点击此处。

场景光照。 定义光源的数量及其位置以在 3D 窗口中照亮模型。 光源的相对位置显示在列表右侧。 所有光源都随查看器移动。

显示导览立方体。 在 3D 窗口中显示或隐藏导览立方体。导览立方体有三种不同类型的热点区域：边、拐角和面。当您将指针置于导览立方体的边、拐角或面上时，相应的区域将高亮显示。通过这些区域，可以在标准视图之间切换，也可进行旋转。

显示曲面边。如果选中，则在 3D 窗口中显示曲面边界。

反锯齿.激活用于最小化 3D 窗口中变形构件的技术。

环境遮蔽。激活 3D 窗口中的周围介质遮挡着色和渲染技术。

选择模式.指定要在 3D 窗口中选择的对象表示。

用颜色高亮显示 - 高亮显示所选内容。

透视所选对象 - 除所选对象外，其余对象呈灰色半透明状态。

半透明壳 - 整个模型的外壳呈半透明状态，非选定的内部对象将被隐藏。

描画光标悬停对象。所选内容带轮廓显示。

高亮显示   当选择或指向 3D 窗口中的项或 2D 窗口中的标注时，它会高亮显示。所选内容的项号（或 CSN）将以弹出菜单的形式显示。目录编创者可以更改颜色方案。默认情况下会应用以下颜色方案：

黄色 — 表示选定的零组件或标注。对应的 DPL 行也会在 DPL 表格中处于选中和高亮显示状态。

绿色 - 鼠标指针所指向的标注。对应的 DPL 行也会在 DPL 表格中处于高亮显示状态。

3D 视图条

用于控制 3D 窗口中的查看模式的按钮。

用户界面 按钮 用途
设置中心 用于指定 3D 模型的旋转中心。
显示旋转中心 用于隐藏或显示旋转中心。
全部适合 使整个 3D 模型在 3D 窗口中完全可见。
水平对齐 将摄像机的水平（但不是纵向）轴设为与坐标系中的水平平面平行。
显示坐标轴 显示或隐藏 3D 窗口中的方位轴。
测量 打开工具以测量距离。 在 3D 窗口中，只需在一个点上单击一次，然后在第二个点上再次单击。 然后将显示两个红点之间的距离。 要获得投影到竖直面或水平面上的点之间的距离，从列表中选择投影平面名称。 要更改长度单位，选择单位。 要设置指引线长度，拖动 H 滑块。 选择保持可见，使尺寸线在关闭测量工具后仍然可见。要设置新的点，单击清除或按 Esc 键，然后在 3D 窗口中指定点。

WARNING

切勿在工作流程中直接使用测量结果，因为 3D 插图可能与实际产品有所不同。要获取制造或维修所需的尺寸，请参考相关技术文档。

截平面 显示模型，就好像它是按平面切割的一样。 使用控件设置（左视图、俯视图、前视图）和旋转（X 和 Y）截平面。要移动截平面，拖动 D 滑块。 使用翻转镜像截面。 选择精确模式可提高滑块的精度。选择保持可见，使截面在关闭工具后可见。要将截平面移至所选内容的中心，单击重置。

有多个支持从指定位置显示 3D 模型的查看模式：

用户界面 按钮 用途
前部 从正面显示模型。
后退 从背面显示模型。
向左 从左侧显示模型。
向右 从右侧显示模型。
置顶 从顶部显示模型。
置底 从底部显示模型。
正等测 在正等测图中显示模型。
选择条

用于显示和隐藏在 3D 窗口中选择的项，以及使其适合窗口和变得透明。

用户界面 用途
在单独的窗口中显示选定对象 如果启用，则在辅助 3D 窗口中显示选择内容。
忽略透明度 如果启用，就可以穿过 3D 窗口中的透明项执行单击操作，因为透明项将不对用户的单击做出响应。
全部恢复 显示所有隐藏和半透明对象。
适合窗口 使所选项的限定框适合 3D 窗口。
显示 使先前在 3D 窗口中隐藏的所选项可见。对应 DPL 行的复选框将处于选中状态。
隐藏 隐藏 3D 窗口中的所选项。对应 DPL 行的复选框将处于取消选中状态。
全不选 取消选择所有项。
透明度 用于设置所选项的透明度值。如果在 3D 窗口中选择一个或多个项或者在 DPL 表格中选择一个或多个 DPL 行，则透明度命令可用。
DPL 表格窗口

显示 DPL 表格。注意，DPL 表条可能包含一组用于按选定准则过滤 DPL 数据的过滤器（例如：UCA、模型）。使用一个还是另一个过滤器取决于创建此 IPC 所使用的信息标准。

用户界面 按钮 用途
打印表格 用于打印 DPL 表格。
复合打印 用于打印 IPC 中可用的 DPL 表和所有 2D 图像。如果选中完整表，则打印完整的 DPL 表格和所有插图。如果取消选中完整表，则仅打印当前 DPL 工作表和对应的插图。每个 2D 图像（插图）均打印在单独的页面上。
完整表   显示整个 DPL 表格。与当前工作表没有关联的行（非活动行）的文本为灰色。
导出至 CSV 将所有选定 DPL 行的数据导出至逗号分隔文件。如果未选择任何 DPL 行，则仅导出与可见 3D 项相关联的 DPL 行。
帮助 提供用户界面的帮助系统。
标题区   “标题”区提供有关 IPC 的一些信息，例如：出版物名称、工作表名称、发布日期。
复选框列   复选框列是具有勾选框的 DPL 列。将使用以下指示：

— DPL 行在当前工作表的 3D 窗口中具有对应的可见项。

— 表示对应的项会隐藏在当前工作表的 3D 窗口中。

若无复选框，则表示 DPL 行没有针对当前工作表的对应 3D 项。
高亮显示   当选择或指向 DPL 行时，它会高亮显示。高亮显示的 DPL 行的项编号（或 CSN）将以弹出窗口的形式显示。目录编创者可以更改颜色方案。默认情况下会应用以下颜色方案：

碧绿 - 鼠标指针所指向的 DPL 行，是指附图的项。项和标注也会在 3D/2D 窗口中高亮显示。

黄色 — 表示选定的 DPL 行。对应的项和标注也会在 3D/2D 窗口中高亮显示。

粉红色 - 鼠标指针所指向的 DPL 行，但其中没有附图的项。

3D/2D 和 “DPL 表”窗口中的上下文相关菜单

右键单击 3D 窗口中的某个零组件或 DPL 表中的某一 DPL 行或 2D 窗口中的某个标注，即可访问上下文相关菜单。

用户界面 含义
项/CSN 显示选定项或 DPL 行或标注的项编号或目录序列号 (CSN)。
适合窗口 使所选项的限定框适合 3D 窗口。
显示/隐藏 显示或隐藏 3D 窗口中的选定项。对应 DPL 行的复选框将分别处于选中或取消选中状态。
选择/取消选择 选择或取消选择 3D 窗口中的选定项或 DPL 表格中的选定 DPL 行或 2D 窗口中的选定标注。
工作表列表 用于从可用工作表列表中选择一个工作表来说明所选的 DPL 行。建议用于 DPL 行的工作表以粗体显示。当前工作表已使用复选标记符号进行标记。
全部显示 使所有隐藏的项在 3D 窗口中可见并在 DPL 表格中选中对应的复选框。
放大 增大图像。
缩小 减小图像。
最适合 使图像适合边缘边界。
关于 显示 IPC、3D 查看器和 2D 查看器的版本。
DPL 表 - 主要流程
操作 用户操作
选择 DPL 行 单击所需的 DPL 行。

DPL 行和对应的 3D 项（或 2D 窗口中的标注）将处于高亮显示状态。
逐个选择多个 DPL 行 按住 Ctrl 键，然后单击每个所需的 DPL 行。
选择一组连续的 DPL 行 单击第一个 DPL 行，按住 Shift 键，然后单击最后一个 DPL 行。
使项适合 3D 窗口 双击所需的 DPL 行。
在 3D 窗口中隐藏项 取消选中所需 DPL 行的复选框。
在 3D 窗口中显示项 选中所需 DPL 行的复选框。
在 3D 窗口中隐藏/显示一组项 选择要隐藏/显示其对应项的多个 DPL 行，然后使用“选择”条上的“隐藏”和“显示”按钮。
2D 窗口 - 主要流程

2D 窗口中的标注是允许项和对应 DPL 行之间进行交互的热点。右键单击 2D 窗口可访问可用的上下文相关命令。右键单击某个标注可访问与零组件有关的上下文相关命令（这些命令如“3D/2D 窗口和 DPL 表中的上下文相关菜单”部分中所述）

操作 用户操作
选择项标注 单击所需标注。
平移 移动鼠标时按住鼠标左键。
缩放 向前或向后旋转鼠标滚轮。
3D 窗口 - 主要流程
操作 用户操作
选择项 单击所需项。
逐个选择多个项 按住 Ctrl 键，然后单击每个所需的 3D 项。
放大/缩小 向前或向后旋转鼠标滚轮。在 Apple 计算机上，使用控制修改键。
平移 按住鼠标滚轮并移动鼠标。在 Apple 计算机上，使用选项修改键。
自转 移动鼠标时按住鼠标左键。
设置旋转中心 在“3D 视图”条上，单击设置中心 ，然后在 3D 窗口中单击所需的项。

或者按住 ALT 键，然后在 3D 窗口中单击任意项。
使 3D 模型适合窗口 在“3D 视图”条上，单击适合 。
适合所选项 在 3D 窗口或 DPL 表中选择所需的项，然后单击“选择”条上的适合。
显示指定位置的 3D 模型 在“3D 视图”条上，单击所需的按钮（ 前视图、 后视图、 左视图、 右视图、 俯视图、 仰视图和 正等测图）。
隐藏/显示一组项 按住 Ctrl 键，单击每个所需的 3D 项，然后使用“选择”条上的“隐藏”和“显示”按钮。
设置透明度/不透明度 在 3D 窗口中选择所需的项，然后拖动“选择”条上的透明度滑块，直到在 3D 窗口中获得所需的效果。
