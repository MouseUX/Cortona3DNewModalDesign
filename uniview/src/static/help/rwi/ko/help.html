작업 지침 - 사용자 인터페이스

대화형 문서는 다음과 같은 두 가지 모드에서 사용할 수 있습니다.

미리보기 모드

  • 전체 절차가 재생됩니다.
  • 사용자는 절차 재생을 제어할 수 있습니다.
  • 사인오프 명령이 필요하지 않을 경우 사용할 수 없습니다.

작업 모드

  • 현재 타스크가 재생됩니다.
  • 사용자는 현재 타스크의 재생을 제어할 수 있습니다.
  • 사용자는 사인오프 버튼을 클릭하여 다음 타스크로 이동해야 합니다.

대화형 문서의 사용자 인터페이스는 데이터 윈도우 및 3D/2D 윈도우의 두 작업 영역으로 구성됩니다. 각 작업 영역에는 자체 제어 요소가 있습니다.

도움말을 보려면 대화형 문서의 오른쪽 하단 모서리에 있는 도움말 버튼을 누르십시오.

데이터 윈도우

데이터 윈도우에는 제목 영역과 필수항목, BOM, 리소스, 지침 및 문서 탭이 포함되어 있습니다.

제목 영역

제목 영역에는 작업 지침에 대한 일반 정보가 표시됩니다. 예를 들어, 작업 코드, 작업 유형, 예상 시간.

필수항목 탭

필수항목 탭은 기술 절차를 시작하기 전에 충족되어야 하는 조건을 설명합니다. 대화형 문서를 처음 열 때 탭이 활성화됩니다.

BOM 탭을 엽니다

BOM(Bill of Material) 탭에는 들여쓰기된 파트 리스트가 표시됩니다. 각 BOM 레코드는 파트 번호, 설명, 수량 및 대체 파트 번호에 대한 다음 정보를 제공합니다.

BOM 레코드를 클릭하면 3D 윈도우에서 해당 아이템이 맞춰지고 강조 표시됩니다.

BOM 레코드 위에 포인터를 배치하면 3D 윈도우에서 레코드 및 해당 아이템이 강조 표시됩니다.

사용자는 더하기 기호를 클릭하면 계층이 확장됩니다.

리소스 탭

리소스 탭은 기술 절차를 수행하는 데 필요한 리소스에 대해 설명합니다.

지침 탭

지침 탭은 현재 타스크, 필수 파트, 필수 리소스 섹션으로 구성되어 있습니다. 현재 타스크 섹션에는 현재 타스크에 대한 텍스트가 표시됩니다.

텍스트에는 3D 아이템 및 2D 이미지에 대한 하이퍼링크가 포함될 수 있습니다. 3D 아이템에 대한 링크가 포함된 텍스트는 파선으로 밑줄이 그어집니다. 사용자는 밑줄이 그어진 텍스트를 클릭하면 3D 윈도우에서 해당 아이템이 선택되고 맞춰집니다.
2D 아이템에 대한 링크가 포함된 텍스트는 점선으로 밑줄이 그어집니다(예: ). 사용자는 밑줄이 그어진 텍스트를 클릭하면 2D 윈도우에서 해당 이미지가 열립니다.

필수 파트 섹션은 해당 3D 아이템에 대한 링크가 포함된 파트에 대해서만 BOM 데이터를 표시합니다.

필수 리소스 섹션은 현재 타스크에 필요한 리소스가 표시됩니다.

문서 탭

문서 탭에는 작업 지침의 전체 텍스트가 표시됩니다.

작업 모드에서 사인오프 버튼을 클릭하여 지침의 다음 타스크로 이동할 수 있습니다. 이전 타스크는 해당 타스크에 대한 사인오프 버튼을 클릭한 시간 및 날짜를 표시합니다(예: 사인오프: 2010년 7월 21일 수요일, 12:47:10 PM). 이 데이터는 어디에도 저장되지 않으므로 현재 세션 내에서만 시간 및 날짜가 표시될 수 있습니다.

절차를 재생하는 동안 검사 필요 메시지 상자가 나타날 수 있습니다. 해당 타스크는 검사 필요 메시지 상자에서 닫기 버튼을 클릭한 시간 및 날짜를 표시합니다(예: 검사됨: 2010년 7월 21일 수요일, 12:48:57 PM). 시간 및 날짜는 현재 세션 내에서만 표시 가능합니다.

미리 보기 모드에서 절차의 필수 타스크를 클릭하면 문서 탭에 타스크가 강조 표시되어 절차 재생이 활성화됩니다. 작업 모드에서는 현재 타스크만 재생에 사용할 수 있습니다.

문서 텍스트에는 3D 아이템 및 2D 이미지에 대한 하이퍼링크가 포함될 수 있습니다(지침 탭 참조).

3D 윈도우

3D 윈도우는 문서 탭에 설명된 절차의 3D 표현을 제공합니다. 3D 윈도우에서 재생 바를 사용하여 절차를 재생하고 마우스 버튼을 사용하여 탐색할 수 있습니다. 3D 윈도우의 오른쪽 상단 모서리에 있는 로케이터(예: 축 삼각대)를 사용하여 단위 또는 파트의 공간 위치를 확인할 수 있습니다.

재생 바

3D 윈도우에서 절차 재생을 제어하기 위한 버튼입니다.

명령 목적
재생 중지 또는 일시 중지될 경우 절차의 재생을 시작합니다.
일시 중지 현재 단계에서 재생을 중지합니다.
정지 현재 단계를 중지하고 처음으로 되감습니다.
사인오프 작업 모드에서 사용할 수 있습니다.
다음 타스크를 활성화합니다. 타스크는 문서 탭에서 강조 표시되고 절차의 해당 모멘트는 3D 윈도우에서 재생됩니다. 문서 탭의 이전 타스크는 해당 타스크에 대해 사인오프 버튼을 클릭한 시간 및 날짜를 표시합니다.

설정

설정 리스트를 표시하거나 숨깁니다.

속도. 재생 속도를 설정합니다.

시점 동결. 선택된 경우 카메라를 현재 뷰로 잠그므로 절차를 재생하는 동안 카메라의 이동이 방지됩니다.

알림 메시지 비활성화. 선택된 경우 재생 중에 절차의 알림 메시지가 비활성화됩니다.

PMI. 선택된 경우 PMI 데이터가 표시됩니다. 제품 제조 정보(PMI)는 제품 컴포넌트 또는 하위 시스템을 제조하기 위해 필요한 3D CAD에서 비지오메트리 특성을 나타냅니다.

장면 조명. 3D 윈도우에서 모델을 비출 광원의 수와 위치를 정의합니다. 광원의 상대 위치가 리스트의 오른쪽에 표시됩니다. 모든 광원이 뷰어와 함께 이동됩니다.

탐색 육면체를 표시합니다. 3D 윈도우에서 탐색 육면체를 표시하거나 숨깁니다. 탐색 육면체에는 세 가지 유형의 핫스팟 영역(모서리, 코너 및 면)이 있습니다. 탐색 육면체의 모서리, 코너 또는 면에 포인터를 놓으면 해당 영역이 강조 표시됩니다. 이러한 영역을 사용하여 표준 뷰 간에 전환하고 회전을 수행할 수 있습니다.

곡면 모서리 표시. 선택된 경우 3D 윈도우에서 곡면 경계의 디스플레이를 활성화합니다.

앤티앨리어싱. 3D 윈도우에서 왜곡 아티팩트를 최소화하는 기법을 활성화합니다.

앰비언트 오클루전. 3D 윈도우에서 주변 폐색 음영처리 및 렌더링 기법을 활성화합니다.

선택 모드. 3D 윈도우에서 선택을 위한 개체 표현을 지정합니다.

색상으로 강조 표시 - 선택이 강조 표시됩니다.

X선 선택한 개체 - 선택 사항을 제외한 개체는 회색 및 반투명하게 표시됩니다.

반투명 셸 - 전체 모델의 외부 셸은 반투명으로 표시되고 선택되지 않은 내부 개체는 숨겨집니다.

호버링된 개체 윤곽선 그리기. 선택 영역의 윤곽선이 그려집니다.

재생 진행률 표시줄

재생 진행률을 표시합니다. 바를 클릭하면 원하는 절차 모멘트를 선택할 수 있습니다.

선택 바

3D 윈도우에서 선택된 개체를 맞춤, 표시, 숨기기 및 투명하게 만들기 위한 버튼입니다.

사용자 인터페이스 버튼 목적
별도의 윈도우에 선택한 개체 표시 이 옵션을 활성화하면 보조 3D 윈도우에 선택이 표시됩니다.
투명도 무시 이 옵션을 활성화하면 투명 아이템이 사용자 클릭에 둔감해지므로 3D 윈도우에서 투명 아이템을 클릭하여 연결할 수 있습니다.
모두 복원 숨겨진 개체와 반투명 개체를 모두 보여줍니다.
맞춤 선택한 아이템의 경계 상자를 3D 윈도우에 맞춥니다.
표시 3D 윈도우에서 이전에 숨긴 선택한 아이템을 표시합니다.
숨기기 3D 윈도우에서 선택한 아이템을 숨깁니다.
아무 것도 선택하지 않습니다 모든 아이템을 선택 취소합니다.
투명도   선택된 아이템에 대한 투명도 값을 설정할 수 있습니다. 3D 윈도우에서 하나 이상의 아이템이 선택된 경우 투명도 명령을 사용할 수 있습니다.

보조 3D 윈도우

선택한 개체가 보조 3D 윈도우에 표시됩니다.

사용자 인터페이스 버튼 목적
중심 설정 3D 모델에 대한 회전 중심을 지정할 수 있습니다.
회전 중심 표시 회전 중심을 숨기거나 표시할 수 있습니다.
전체 맞춤 3D 윈도우에서 전체 3D 모델을 완전히 표시되도록 만듭니다.
수평으로 정렬 카메라의 수평(세로 방향이 아님) 축을 좌표계의 수평 평면에 평행하게 배치합니다.
측정 도구를 열어 거리를 측정합니다. 3D 윈도우에서 한 점을 한 번 클릭한 다음 두 번째 점을 다시 클릭합니다. 그러면 빨간색 점 두 개 사이의 거리가 표시됩니다. 수직면 또는 수평면에 투영된 두 점 사이의 거리를 확인하려면 리스트에서 투영면 이름을 선택합니다. 길이 단위를 변경하려면 단위를 선택합니다. 지시선 길이를 설정하려면 H 슬라이더를 끌어옵니다. 측정 도구를 닫은 후 치수선을 보이게 하려면 계속 표시를 선택합니다. 새 점을 설정하려면 지우기를 클릭하거나 Esc 키를 누르고 3D 윈도우에서 점을 지정합니다.

WARNING

3D 그림이 실제 제품과 다를 수 있으므로 측정 결과를 직접 워크플로에 사용하지 마십시오. 제조 또는 서비스에 필요한 치수를 얻으려면 관련 기술 문서를 참조하십시오.

절단면 모델을 평면에 의해 절단된 것처럼 표시합니다. 제어를 사용하여 (좌측, 상단, 정면)을 설정하고 (X, Y) 절단면을 회전합니다. 절단면을 이동하려면 D 슬라이더를 끌어옵니다. 뒤집기를 사용하여 단면을 미러링합니다. 슬라이더의 정확도를 높이려면 정밀 모드를 선택합니다. 도구를 닫은 후 단면을 보이게 하려면 계속 표시를 선택합니다. 절단면을 선택 영역의 중앙으로 이동하려면 재설정을 클릭합니다.

3D 윈도우의 상황에 맞는 메뉴

3D 윈도우의 아무 곳이나 마우스 오른쪽 버튼으로 클릭하여 상황에 맞는 메뉴에 액세스합니다.

명령 목적
재설정 단계 뷰를 재설정합니다.
전체 맞춤 아이템이 3D 윈도우에 완전히 표시되도록 만듭니다.
정보 대화형 문서, 3D 뷰어 및 2D 뷰어의 버전을 표시합니다.
2D 윈도우

2D 윈도우에는 데이터 윈도우에 설명된 절차와 연결된 벡터 및 래스터 이미지가 표시됩니다.

사용자는 데이터 윈도우에서 2D 이미지 하이퍼링크를 클릭하면 2D 윈도우가 3D 윈도우를 대체합니다(지침 탭 또는 문서 탭).

2D 윈도우에서 마우스 오른쪽 버튼을 클릭하여 사용 가능한 상황에 맞는 명령에 액세스합니다. 명령 리스트는 2D 윈도우에서 사용 중인 2D 뷰어에 따라 다를 수 있습니다.
