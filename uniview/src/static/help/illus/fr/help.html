Illustration – interface utilisateur

L'interface utilisateur de l'illustration interactive contient deux espaces de travail : 3D et 2D. Chaque espace de travail possède ses propres éléments de commande décrits ci-dessous. Toutes les couleurs utilisées pour l'interface utilisateur dans ce document sont les couleurs par défaut.

L'illustration peut se trouver dans les représentations suivantes :

  • 2D et 3D permet de visualiser des images 2D et des illustrations 3D. Dans ce cas, l'illustration interactive a un bouton (Graphique 2D ) pour basculer entre deux fenêtres.
  • 3D seulement permet de n'afficher que les images 3D de l'illustration interactive. Dans ce cas, les images 2D ne sont pas disponibles.
  • 2D seulement permet de n'afficher que les images 2D de l'illustration interactive. Dans ce cas, l'illustration 3D n'est pas disponible.
Fenêtre 3D/2D

La fenêtre 3D affiche une image 3D de la feuille active. Vous pouvez naviguer dans la fenêtre 3D à l'aide de la souris ou du clavier. Un positionneur (trièdre d'axe) dans le coin supérieur droit de la fenêtre 3D permet de déterminer l'orientation du modèle 3D par rapport aux axes.

La fenêtre 2D affiche des images vectorielles et tramées associées à la feuille active.

Interface utilisateur Bouton Objectif
Feuille   Permet de sélectionner la feuille détaillée dans la liste des feuilles disponibles.
Réinitialiser Rétablit la vue initiale sur la feuille active.
Graphiques 2D Permet de basculer entre les fenêtres 2D et 3D. Le bouton n'est disponible qu'en mode d'affichage 2D et 3D.
Impression Graphique Permet d'imprimer l'image 3D/2D active telle qu'elle s'affiche dans la fenêtre 3D/2D.
Ignorer transparence (pour la fenêtre 3D) Si cette option est activée, permet de cliquer dans les objets transparents dans la fenêtre 3D, car les objets transparents ne réagissent plus au clic de l'utilisateur.
Tout rétablir (pour la fenêtre 3D) Affiche tous les objets masqués et semi-transparents.
Omettre l'animation (pour fenêtre 3D) Si cette option est activée, permet d'éviter une transition fluide (animation) d'une feuille à l'autre et rend la transition entre les feuilles instantanée.
Paramètres

Afficher ou masque la liste des paramètres.

Vitesse d'animation de transition. Ajuste une vitesse de lecture pour la transition de l'explosion.

Activer le contrôle Animation de transition. Si sélectionné, active le curseur qui vous permet de vérifier manuellement les transitions entre les feuilles. Pour afficher le curseur, déplacez le pointeur de la souris au-dessus du bas de la fenêtre 3D, ou tapez en bas de la fenêtre si vous avez un écran tactile.

Eclairage de scène. Définit le nombre de sources de lumière et leur emplacement pour éclairer le modèle dans la fenêtre 3D. Les positions relatives des sources de lumière sont affichées à droite de la liste. Toutes les sources de lumière se déplacent avec l’afficheur.

Afficher le cube de navigation. Affiche ou masque le cube de navigation dans la fenêtre 3D. Le cube de navigation comporte trois types de zones de HotSpot : arête, coin et face. Lorsque vous positionnez le pointeur sur l'arête, le coin ou la face du cube de navigation, la zone correspondante est mise en surbrillance. Avec ces zones, vous pouvez basculer entre les vues standard et effectuer des rotations.

Afficher les arêtes de surface. Si sélectionné, affiche les contours de surface dans la fenêtre 3D.

Anticrénelage. Active une technique pour minimiser les artefacts de distorsion dans la fenêtre 3D.

Occlusion ambiante. Active l'ombrage de l'occlusion ambiante et la technique de rendu dans la fenêtre 3D.

Mode de sélection. Spécifie la représentation des objets pour la sélection dans la fenêtre 3D.

Contraster avec une couleur – la sélection est mise en surbrillance.

Radiographier les objets sélectionnés – les objets apparaissent gris et translucides, à l'exception de la sélection.

Coque transparente – la coque externe du modèle entier apparaît translucide, les objets internes non sélectionnés sont masqués.

Entourer les objets survolés. La sélection est entourée.

Aide Fournit un système d'aide pour l'interface utilisateur.
Mise en surbrillance   Lorsque vous sélectionnez un objet ou que vous pointez vers un objet dans la fenêtre 3D ou une légende dans la fenêtre 2D, il est mis en surbrillance. Le schéma de couleurs peut être modifié par l'auteur de l'illustration. Le schéma de couleurs suivant est appliqué par défaut :

Jaune — pour l'objet sélectionné.

Vert — pour la légende vers laquelle vous pointez.
Barre de vue 3D

Boutons de commande des modes d'affichage dans la fenêtre 3D.

Interface utilisateur Bouton Objectif
Définir le centre Permet de spécifier le centre de rotation du modèle 3D.
Afficher le centre de rotation Permet d'afficher ou de masquer le centre de rotation.
Ajuster tout Permet de rendre l'ensemble du modèle 3D entièrement visible dans la fenêtre 3D.
Aligner l'horizon Positionne l'axe horizontal (mais pas longitudinal) de la caméra parallèlement au plan horizontal dans le système de coordonnées.
Afficher les axes de coordonnées Affiche ou masque les axes d'orientation dans la fenêtre 3D.
Mesurer Ouvre l'outil permettant de mesurer une distance. Dans la fenêtre 3D, cliquez simplement une fois sur un point, puis cliquez à nouveau sur le deuxième point. La distance entre les deux points rouges doit ensuite s'afficher. Pour obtenir la distance entre les points projetés sur le plan vertical ou horizontal, sélectionnez le nom du plan de projection dans la liste. Pour modifier l'unité de longueur, sélectionnez Unités. Pour définir la longueur des lignes de référence, faites glisser le curseur H. Sélectionnez Garder visible pour rendre la ligne de dimension visible après avoir fermé l'outil de mesure. Pour définir de nouveaux points, cliquez sur Effacer ou appuyez sur la touche Echap, puis spécifiez des points dans la fenêtre 3D.

WARNING

N'utilisez pas les résultats de mesure directement dans le traitement car l'illustration 3D peut différer du produit réel. Pour obtenir les dimensions nécessaires à la fabrication ou à la maintenance, reportez-vous à la documentation technique appropriée.

Plan de section Affiche le modèle comme s'il avait été coupé par un plan. Utilisez les contrôles pour définir (Gauche, Haut, Avant) et faire pivoter (X et Y) le plan de section. Pour déplacer le plan de section, faites glisser le curseur D. Utilisez Inverser pour mettre en miroir la section. Sélectionnez le Mode Précis pour améliorer la précision des curseurs. Sélectionnez Garder visible pour rendre la section visible après avoir fermé l'outil. Pour déplacer le plan de section vers le centre de la sélection, cliquez sur Réinitialiser.

Interface utilisateur Objectif

Fenêtre 2D – principales procédures

Cliquez avec le bouton droit dans la fenêtre 2D pour accéder aux commandes contextuelles disponibles.

Opération Action Utilisateur

Panoramique Déplacez la souris tout en maintenant le bouton de gauche enfoncé.

Ou en maintenant la touche Alt enfoncée, cliquez sur un objet dans la fenêtre 3D.
