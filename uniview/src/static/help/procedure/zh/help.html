流程 - 用户界面

用户界面由两个工作区组成：“文档”和“3D/2D”。每个工作区都有其自己的控制元素，如下所述。注意，用户界面的外观取决于用于创建此交互式流程的信息标准。

文档窗口

“文档”窗口显示根据指定信息标准创建的文档。文档将提供员工执行任务的分步流程。流程步骤可以包含一些交互式元素。单击超链接可查看 3D 窗口中的当前步骤或 2D 窗口中的 2D 图像。

按钮 描述
过滤器 根据基于 DITA 文档的流程的条件处理属性进行过滤。根据基于 S1000D 数据模块的流程的适用性进行过滤。
3D 窗口

3D 窗口提供文档窗口中所述流程的 3D 表示。在 3D 窗口中，可以使用“回放”条上的按钮播放流程，使用鼠标进行导览。3D 窗口右上角的定位符（例如，三轴架）可用于确定单元或零件的空间位置。

如果第一次打开流程时 3D 窗口处于关闭状态，可单击“文档”窗口中相应的控制元素。

回放条

用于控制 3D 窗口中流程回放的按钮。

命令 用途
播放 在停止或暂停时开始流程的回放。
暂停 在当前步骤处停止回放。
上一个 返回上一步。
下一个 移至流程的下一步。
设置 显示或隐藏设置列表。

速度。设置回放速度。

冻结视点。如果选中，则会将摄像机锁定到当前视图，从而防止在流程回放时摄像机移动。

一步式回放。如果选中，则锁定当前步骤的回放。

禁用警报消息。如果选中，则抑制回放期间向您通知有关流程警报的消息。

显示内嵌注释。如果选中，则在 3D 窗口中显示流程注释。

场景光照。 定义光源的数量及其位置以在 3D 窗口中照亮模型。 光源的相对位置显示在列表右侧。 所有光源都随查看器移动。

显示导览立方体。 在 3D 窗口中显示或隐藏导览立方体。导览立方体有三种不同类型的热点区域：边、拐角和面。当您将指针置于导览立方体的边、拐角或面上时，相应的区域将高亮显示。通过这些区域，可以在标准视图之间切换，也可进行旋转。

显示曲面边。如果选中，则在 3D 窗口中显示曲面边界。

反锯齿.激活用于最小化 3D 窗口中变形构件的技术。

环境遮蔽。激活 3D 窗口中的周围介质遮挡着色和渲染技术。

选择模式.指定要在 3D 窗口中选择的对象表示。

用颜色高亮显示 - 高亮显示所选内容。

透视所选对象 - 除所选对象外，其余对象呈灰色半透明状态。

半透明壳 - 整个模型的外壳呈半透明状态，非选定的内部对象将被隐藏。

描画光标悬停对象。所选内容带轮廓显示。

回放进度条 进度条指示流程的相对进度。拖动或单击进度条以查找所需的流程步骤。
帮助 提供用户界面的帮助系统。
展开，折叠 相对于查看器窗口框架展开或折叠 3D 窗口。

选择条

用于显示和隐藏在 3D 窗口中选择的对象并使其适合窗口和变得透明的按钮。

用户界面 用途
在单独的窗口中显示选定对象 如果启用，则在辅助 3D 窗口中显示选择内容。
忽略透明度 如果启用，就可以穿过 3D 窗口中的透明项执行单击操作，因为透明项将不对用户的单击做出响应。
全部恢复 显示所有隐藏和半透明对象。
适合窗口 使所选项的限定框适合 3D 窗口。
显示 使先前在 3D 窗口中隐藏的所选项可见。
隐藏 隐藏 3D 窗口中的所选项。
全不选 取消选择所有项。
透明度 用于设置所选项的透明度值。如果在 3D 窗口中选择一个或多个项，则透明度命令可用。

在 3D 窗口中导览

操作 用户操作
放大/缩小 向前或向后旋转鼠标滚轮。在 Apple 计算机上，使用控制修改键。
平移 按住鼠标滚轮并移动鼠标。在 Apple 计算机上，使用选项修改键。
自转 移动鼠标时按住鼠标左键。
设置旋转中心 按住 ALT 键，单击 3D 窗口中的任意项，或单击设置中心 ，然后在 3D 窗口中单击所需的项。
显示或隐藏旋转中心 单击显示旋转中心 以在 3D 窗口中显示或隐藏旋转中心及其轴。
全部适合 单击全部适合显示 ，使项在 3D 窗口中完全可见。
水平对齐 单击水平对齐 ，将摄像机的水平（但不是纵向）轴设为与坐标系中的水平面平行。
显示或隐藏坐标轴 单击显示坐标轴 以显示或隐藏 3D 窗口中的方位轴。
测量距离 单击测量 以打开测量工具。 在 3D 窗口中，只需在一个点上单击一次，然后在第二个点上再次单击。 然后将显示两个红点之间的距离。 要获得投影到竖直面或水平面上的点之间的距离，从列表中选择投影平面名称。 要更改长度单位，选择单位。 要设置指引线长度，拖动 H 滑块。 选择保持可见，使尺寸线在关闭测量工具后仍然可见。要设置新的点，单击清除或按 Esc 键，然后在 3D 窗口中指定点。

WARNING

切勿在工作流程中直接使用测量结果，因为 3D 插图可能与实际产品有所不同。要获取制造或维修所需的尺寸，请参考相关技术文档。

剖视图 单击截平面 ，可使模型显示为如同被平面剖切后一样。使用控件设置（左视图、俯视图、前视图）和旋转（X 和 Y）截平面。要移动截平面，拖动 D 滑块。 使用翻转镜像截面。 选择精确模式可提高滑块的精度。选择保持可见，使截面在关闭工具后可见。要将截平面移至所选内容的中心，单击重置。
以不同视角显示 3D 窗口中的对象：前视图、后视图、左视图、右视图、俯视图、仰视图和正等测图。单击按钮激活视图。

3D 窗口中的上下文相关菜单

在 3D 窗口中右键单击任意位置以访问上下文相关菜单。

命令 用途
重置 重置步骤视图。
全部适合 使项在 3D 窗口中完全可见。
关于 显示流程、3D 查看器和 2D 查看器的版本。
2D 窗口

2D 窗口显示与文档窗口中所述流程关联的矢量图像和光栅图像。当用户在文档窗口中单击 2D 图像超链接时，2D 窗口将替换 3D 窗口。右键单击 2D 窗口可访问可用的上下文相关命令。注意，命令列表取决于 2D 窗口中使用的 2D 查看器。

在 2D 窗口中导览

操作 用户操作
平移 移动鼠标时按住鼠标左键。
缩放 向前或向后旋转鼠标滚轮。

2D 窗口中的按钮

按钮 描述
3D 打开 3D 窗口。
展开，折叠 相对于查看器窗口框展开或折叠 2D 窗口。
下一页，上一页 对于多页插图，可导航至下一页或上一页。
