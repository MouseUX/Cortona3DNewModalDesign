Procédure – interface utilisateur

L'interface utilisateur se compose de deux espaces de travail : Document et 3D/2D. Chaque espace de travail possède ses propres éléments de commande décrits ci-dessous. Veuillez noter que l'apparence de l'interface utilisateur dépend du standard d'information utilisé pour créer cette procédure interactive.

Fenêtre du document

La fenêtre Document affiche un document créé conformément au standard d'information spécifié. Le document fournit une procédure pas à pas pour qu'un employé exécute une tâche. Les étapes de procédure peuvent contenir des éléments interactifs. Cliquez sur le lien hypertexte pour voir l'étape en cours dans la fenêtre 3D, ou une image 2D dans la fenêtre 2D.

Bouton Description
Filtre Filtrage par attribut de traitement conditionnel pour les procédures basées sur les documents DITA. Filtrage par applicabilité pour les procédures basées sur les modules de données S1000D.
Fenêtre 3D

La fenêtre 3D fournit une représentation 3D de la procédure décrite dans la fenêtre Document. Dans la fenêtre 3D, vous pouvez jouer la procédure à l'aide des boutons de la barre Rejouer et naviguer à l'aide de la souris. Le positionneur (par exemple, le trièdre d'axes) dans le coin supérieur droit de la fenêtre 3D vous permet de déterminer la position spatiale de l'unité ou de la pièce.

Si la fenêtre 3D est fermée lorsque vous ouvrez la publication pour la première fois, cliquez sur l'élément de contrôle correspondant dans la fenêtre Document.

Barre Rejouer

Boutons de contrôle de la lecture de la procédure dans la fenêtre 3D.

Commande Objectif
Lire Démarre la lecture de la procédure si elle est arrêtée ou suspendue.
Pause Arrête la lecture à l'étape courante.
Précédent Recule à l'étape précédente.
Suivant Avance à l'étape suivante de la procédure.
Paramètres Afficher ou masque la liste des paramètres.

Vitesse. Définit la vitesse de lecture.

Geler point de vue. Si sélectionné, verrouille la caméra sur la vue courante, empêchant ainsi le mouvement de la caméra pendant que la procédure est rejouée.

Rejouer en une étape. Si sélectionné, verrouille la lecture de l'étape en cours.

Désactiver les messages d'alerte. Si sélectionné, supprime les messages qui vous avertissent des alertes de la procédure pendant qu'elle est rejouée.

Afficher Commentaires en contexte. Si sélectionné, les commentaires de la procédure sont affichés dans la fenêtre 3D.

Eclairage de scène. Définit le nombre de sources de lumière et leur emplacement pour éclairer le modèle dans la fenêtre 3D. Les positions relatives des sources de lumière sont affichées à droite de la liste. Toutes les sources de lumière se déplacent avec l’afficheur.

Afficher le cube de navigation. Affiche ou masque le cube de navigation dans la fenêtre 3D. Le cube de navigation comporte trois types de zones de HotSpot : arête, coin et face. Lorsque vous positionnez le pointeur sur l'arête, le coin ou la face du cube de navigation, la zone correspondante est mise en surbrillance. Avec ces zones, vous pouvez basculer entre les vues standard et effectuer des rotations.

Afficher les arêtes de surface. Si sélectionné, affiche les contours de surface dans la fenêtre 3D.

Anticrénelage. Active une technique pour minimiser les artefacts de distorsion dans la fenêtre 3D.

Occlusion ambiante. Active l'ombrage de l'occlusion ambiante et la technique de rendu dans la fenêtre 3D.

Mode de sélection. Spécifie la représentation des objets pour la sélection dans la fenêtre 3D.

Contraster avec une couleur – la sélection est mise en surbrillance.

Radiographier les objets sélectionnés – les objets apparaissent gris et translucides, à l'exception de la sélection.

Coque transparente – la coque externe du modèle entier apparaît translucide, les objets internes non sélectionnés sont masqués.

Entourer les objets survolés. La sélection est entourée.

Barre de progression Rejouer La barre de progression indique la progression relative de la procédure. Glissez ou cliquez sur la barre de progression pour rechercher l'étape de procédure requise.
Aide Fournit un système d'aide pour l'interface utilisateur.
Développer/réduire Permet de développer ou de réduire la fenêtre 3D par rapport au cadre de la fenêtre de l'afficheur.

Interface utilisateur Objectif

Navigation dans la fenêtre 3D

Opération Action Utilisateur
Zoom avant/Zoom arrière Tournez la molette de la souris vers l'avant ou l'arrière. Sur les ordinateurs Apple, utilisez la touche modificateur de contrôle.
Panoramique Déplacez la souris avec la molette de la souris enfoncée. Sur les ordinateurs Apple, utilisez la touche modificateur d'option.
Tourner Déplacez la souris tout en maintenant le bouton de gauche enfoncé.
Définir un centre de rotation En maintenant enfoncée la touche ALT, cliquez sur n'importe quel objet dans la fenêtre 3D ou cliquez sur Définir le centre , puis cliquez sur l'objet voulu dans la fenêtre 3D.
Afficher ou masquer le centre de rotation Cliquez sur Afficher le centre de rotation pour afficher ou masquer le centre de rotation et ses axes dans la fenêtre 3D.
Ajuster tout Cliquez sur Ajuster tout pour que les objets soient entièrement visibles dans la fenêtre 3D.
Aligner l'horizon Cliquez sur Aligner l'horizon pour que l'axe horizontal (mais pas longitudinal) de la caméra soit parallèle au plan horizontal dans le système de coordonnées.
Afficher ou masquer les axes de coordonnées Cliquez sur Afficher les axes de coordonnées pour afficher ou masquer les axes d'orientation dans la fenêtre 3D.
Mesure d'une distance Cliquez sur Mesurer pour ouvrir l'outil de mesure. Dans la fenêtre 3D, cliquez simplement une fois sur un point, puis cliquez à nouveau sur le deuxième point. La distance entre les deux points rouges doit ensuite s'afficher. Pour obtenir la distance entre les points projetés sur le plan vertical ou horizontal, sélectionnez le nom du plan de projection dans la liste. Pour modifier une unité de longueur, sélectionnez Unités. Pour définir la longueur des lignes de référence, faites glisser le curseur H. Sélectionnez Garder visible pour rendre la ligne de dimension visible après avoir fermé l'outil de mesure. Pour définir de nouveaux points, cliquez sur Effacer ou appuyez sur la touche Echap, puis spécifiez des points dans la fenêtre 3D.

WARNING

N'utilisez pas les résultats de mesure directement dans le traitement car l'illustration 3D peut différer du produit réel. Pour obtenir les dimensions nécessaires à la fabrication ou à la maintenance, reportez-vous à la documentation technique appropriée.

Une vue en coupe Cliquez sur Plan de section pour afficher le modèle comme s'il avait été coupé par un plan. Utilisez les contrôles pour définir (Gauche, Haut, Avant) et faire pivoter (X et Y) le plan de section. Pour déplacer le plan de section, faites glisser le curseur D. Utilisez Inverser pour mettre en miroir la section. Sélectionnez le Mode Précis pour améliorer la précision des curseurs. Sélectionnez Garder visible pour rendre la section visible après avoir fermé l'outil. Pour déplacer le plan de section vers le centre de la sélection, cliquez sur Réinitialiser.

Commande Objectif
Réinitialiser Réinitialise la vue de l'étape de la procédure.
Ajuster tout Rend les objets entièrement visibles dans la fenêtre 3D.
A propos Affiche les versions de la procédure, de l'afficheur 3D et de l'afficheur 2D.
Fenêtre 2D

La fenêtre 2D affiche des images vectorielles et tramées associées à la procédure décrite dans la fenêtre Document. La fenêtre 2D remplace la fenêtre 3D lorsque l'utilisateur clique sur un lien hypertexte d'image 2D dans la fenêtre document. Cliquez avec le bouton droit dans la fenêtre 2D pour accéder aux commandes contextuelles disponibles. Notez que la liste des commandes dépend de l'afficheur 2D utilisé dans la fenêtre 2D.

Navigation dans la fenêtre 2D

Opération Action Utilisateur
Panoramique Déplacez la souris tout en maintenant le bouton de gauche enfoncé.
Zoom Tournez la molette de la souris vers l'avant ou l'arrière.
